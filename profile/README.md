## How we work

- **Remote dev runs in the browser** at [coder.milton.co](https://coder.milton.co). Sign in with your Milton-Group GitHub account. Workspaces are pre-configured (VS Code, Claude Code, our toolchain) so first-day setup is minutes, not days.
- **Conventions live in `claude-template`** ([repo](https://github.com/Milton-Group/claude-template)). Every Milton repo inherits the same Claude Code baseline and CI shape. See [`docs/`](https://github.com/Milton-Group/claude-template/tree/main/docs) for the full set: ONBOARDING, CONVENTIONS, OPERATING-MODEL, RETROFIT.
- **Infrastructure-as-code lives in `infra`** ([repo](https://github.com/Milton-Group/infra)). Terraform + Packer for Milton's AWS, Cloudflare, GitHub-org, and Coder platforms. Architecture and stack inventory in [`ARCHITECTURE.md`](https://github.com/Milton-Group/infra/blob/main/ARCHITECTURE.md).

## Security

To report a security issue, see [`SECURITY.md`](./SECURITY.md). Critical vulnerabilities should go to `security@milton.co`.

## 我们如何工作

- **远程开发在浏览器中** —— 访问 [coder.milton.co](https://coder.milton.co)。用你的 Milton-Group GitHub 账号登录。工作区预配置好(VS Code、Claude Code、我们的工具链),首日设置只需几分钟。
- **约定保存在 `claude-template`** 中([仓库](https://github.com/Milton-Group/claude-template))。每个 Milton 仓库都继承相同的 Claude Code 基线和 CI 形态。完整文档见 [`docs/`](https://github.com/Milton-Group/claude-template/tree/main/docs):ONBOARDING、CONVENTIONS、OPERATING-MODEL、RETROFIT。
- **基础设施即代码保存在 `infra`** 中([仓库](https://github.com/Milton-Group/infra))。Terraform + Packer 管理 Milton 的 AWS、Cloudflare、GitHub-org 和 Coder 平台。架构与 stack 索引见 [`ARCHITECTURE.md`](https://github.com/Milton-Group/infra/blob/main/ARCHITECTURE.md)。
